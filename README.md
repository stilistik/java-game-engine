# README #

OpenGL based rendering system written in java.

Features:
	-OBJ-FileParser
	-per pixel lighting (diffuse, specular, ambient)
	-flat terrain
	-transparency & fog
	-mipmapping
	-terrain height map
	-terrain collision detection
	-player movement
	-3rd person camera
	-texture atlases
	-gui rendering system
	-multiple light support
